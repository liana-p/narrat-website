# Set Stat

The set\_stat function in narrat sets a HUD stat to a specified value

Syntax: `set_stat [statId] [value]`

See [HUD Stats](../features/hud-stats.md) page for more info on this feature

Example:

```renpy
 set_stat currency 100
```
